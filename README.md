# 📊 Sales Forecasting Dashboard (Power BI)

⭐ If you like this project, feel free to give it a star!

---

## 🔍 Overview
This project is a **Predictive Analytics Dashboard** built using Power BI.  
It analyzes historical sales data and forecasts future trends using time-series forecasting.

---

## 🎯 Objectives
- Analyze historical sales performance  
- Identify trends and patterns  
- Forecast future sales  
- Create an interactive dashboard  

---

## 📈 Key Features

### 📌 KPIs
- Total Sales  
- Total Orders  
- Forecasted Sales (Next Year)  

### 📊 Visualizations
- Sales by Region  
- Sales by Category  
- Sales Trend (Year-wise)  
- Sales Forecast (Predictive Analysis)  

### 🎛 Filters
- Region  
- Category  
- Order Date  

---

## 🔮 Forecasting
Forecasting is applied on **yearly sales data** using Power BI’s built-in time-series feature.

- Forecast Length: **1 Year**  
- Confidence Interval: **95%**

---

## 🛠 Tools & Technologies
- Power BI  
- Data Visualization  
- Predictive Analytics  

---

## 📂 Dataset
Sample Superstore Dataset

---

## 📌 Insights
- Sales show an overall increasing trend  
- Technology category contributes highest revenue  
- Some regions outperform others significantly  
- Forecast predicts continued growth in next year  

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Use filters to explore insights
